# Date picker by raficraft (live demo)

A multilingual date picker from scratch for react.
[code source](https://github.com/raficraft/date_picker_by_raficraft)

## supported language

-  English (EN)
-  English (US)
-  French  (EN)
-  German  (DE)
-  Spanish (SP)
-  Italian (IT)
